# portofolio-adit
Selamat datang di repositori portofolio magang saya. Repositori ini berisi daftar proyek aplikasi yang telah berhasil dikembangkan, dioptimasikan, dan diintegrasikan selama masa perkuliahan.
#1. Proyek: AI Agent (Automation & Chatbot)
Folder AI_Agent berisi implementasi kecerdasan buatan cerdas yang dirancang untuk otomatisasi alur kerja dan integrasi sistem.

Alur Kerja n8n (alur+hasil di n8n): Otomatisasi proses data menggunakan platform workflow automation n8n untuk efisiensi sistem komunikasi data.
Chatbot Website (chatbot_website): Integrasi kecerdasan buatan berbasis LLM yang disematkan langsung pada antarmuka web sebagai asisten virtual interaktif.
#2. Proyek: Sistem Informasi Mading Digital UHN Sugriwa
Folder digital_mading berisi platform portal informasi mading kampus berbasis web yang dikembangkan menggunakan PHP Native dan MySQL untuk manajemen publikasi informasi.

#Fitur Hasil Optimasi :
Sistem Pencarian Cerdas Multi-Kolom: Menggunakan teknik SQL LEFT JOIN berlapis sehingga pencarian mampu mencocokkan kata kunci pada Judul, Deskripsi, Kategori, nama Instansi, hingga rumpun Fakultas sekaligus.
Manajemen Batas Waktu & Deteksi Kadaluarsa: Dilengkapi proteksi input tanggal masa lalu menggunakan JavaScript pada form edit/tambah, serta penanda otomatis teks (Kadaluarsa) secara real-time jika tanggal mading telah terlewati.
Peringkat Terpopuler Real-Time: Sidebar beranda otomatis menyusun peringkat mading 1-5 berdasarkan akumulasi jumlah total pembaca (views) terbanyak yang divalidasi langsung saat detail mading dibuka oleh pengguna.
